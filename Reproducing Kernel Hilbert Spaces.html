1 Espacios de Hilbert con Núcleo Reproducente

Espacio de Hilbert y Núcleo

Un producto interno ⟨u,v⟩ puede ser:

  1. Un producto punto usual: ⟨u,v⟩ = vᵀw = ∑ᵢ vᵢwᵢ
  2. Un producto de núcleo: ⟨u,v⟩ = k(v,w) = ψ(v)ᵀψ(w) (donde ψ(u) puede tener dimensiones infinitas)

Sin embargo, un producto interno ⟨·,·⟩ debe satisfacer:

  1. Simetría
    ⟨u,v⟩ = ⟨v,u⟩ ∀ u,v ∈ 𝒳
  2. Bilinealidad
    ⟨αu + βv, w⟩ = α⟨u,w⟩ + β⟨v,w⟩ ∀ u,v,w ∈ 𝒳, ∀ α,β ∈ ℝ
  3. Definición positiva
    ⟨u,u⟩ ≥ 0, ∀ u ∈ 𝒳
    ⟨u,u⟩ = 0 ⇔ u = 0
Definición. Un Espacio de Hilbert es un espacio de producto interno completo y separable respecto a la norma definida por el producto interno.

Ejemplos:

  1. ℝⁿ con ⟨a,b⟩ = aᵀb
  2. Espacio l₂ de sucesiones sumables cuadráticas: ⟨x,y⟩ = ∑ᵢ₌₁ᵒᵒ xᵢyᵢ
  3. Espacio L₂ de funciones integrables cuadráticas: ⟨f,g⟩ = ∫ₛ f(x)g(x)dx
Definición. k(·,·) es un núcleo reproductor de un espacio de Hilbert ℋ si ∀f ∈ ℋ, f(x) = ⟨k(x,·), f(·)⟩.
===== Página 2 =====

Un Espacio de Hilbert con Núcleo Reproducente (RKHS) es un espacio de Hilbert H con un núcleo reproductor cuyo recorrido es denso en H. Podríamos definir equivalentemente un RKHS como un espacio de Hilbert de funciones con todos los funcionales de evaluación acotados y lineales.

Por ejemplo, el espacio L₂ es un espacio de Hilbert, pero no un RKHS porque la función delta que tiene la propiedad reproductora

f(x) = ∫ₛ δ(x-u)f(u)du

no satisface la condición de integrabilidad cuadrática, es decir,

∫ₛ δ(u)²du ≮ ∞

por lo tanto la función delta no está en L₂.

Definición. k: 𝒳 × 𝒳 → ℝ es un núcleo si:
  1. k es simétrico: k(x,y) = k(y,x)
  2. k es semidefinido positivo: ∀x₁,...,xₙ ∈ 𝒳, la matriz de Gram Kᵢⱼ = k(xᵢ,xⱼ) es semidefinida positiva

Propiedades:

  1. k(x,x) ≥ 0 (Considere la matriz de Gram para n=1)
  2. k(u,v) ≤ √[k(u,u)k(v,v)] (Esta es la desigualdad de Cauchy-Schwarz)

Para ver por qué se cumple la segunda propiedad, consideramos el caso cuando n=2:

Sea a = [k(v,v); -k(u,v)]
K = [k(u,u) k(u,v); k(v,u) k(v,v)] ≽ 0 ⇔ aᵀKa ≥ 0
⇔ [k(v,v)k(u,u) - k(u,v)²]k(v,v) ≥ 0

Por la primera propiedad sabemos k(v,v) ≥ 0, así que k(v,v)k(u,u) ≥ k(u,v)².

Construir un Espacio de Hilbert con Núcleo Reproducente (RKHS)

Dado un núcleo k, defina el "mapeo de características del núcleo reproductor" Φ: 𝒳 → ℝ^𝒳 como:

Φ(x) = k(·,x)

Considere el espacio vectorial:

span{Φ(x): x ∈ 𝒳} = {f(·) = ∑ᵢ₌₁ⁿ αᵢk(·,xᵢ) : n ∈ ℕ, xᵢ ∈ 𝒳, αᵢ ∈ ℝ}

Para f = ∑ᵢ αᵢk(·,uᵢ) y g = ∑ᵢ βᵢk(·,vᵢ), defina ⟨f,g⟩ = ∑ᵢⱼ αᵢβⱼk(uᵢ,vⱼ).

Note que:

⟨f, k(·,x)⟩ = ∑ᵢ αᵢk(x,uᵢ) = f(x)
===== Página 3 =====

Mostramos que ⟨f,g⟩ es un producto interno verificando:

  1. Simetría: ⟨f,g⟩ = ∑ᵢⱼ αᵢβⱼk(uᵢ,vⱼ) = ∑ᵢⱼ βⱼαᵢk(vⱼ,uᵢ) = ⟨g,f⟩
  2. Bilinealidad: ⟨f,g⟩ = ∑ᵢ αᵢg(uᵢ) = ∑ⱼ βⱼf(vⱼ)
  3. Definición positiva: ⟨f,f⟩ = αᵀKα ≥ 0 con igualdad si y solo si f = 0

Del punto 3 también podemos derivar:

  1. ⟨f,g⟩² ≤ ⟨f,f⟩⟨g,g⟩
    Prueba: ∀a ∈ ℝ, ⟨af + g, af + g⟩ = a²⟨f,f⟩ + 2a⟨f,g⟩ + ⟨g,g⟩ ≥ 0. Esto implica que la expresión cuadrática tiene un discriminante no positivo. Por lo tanto, ⟨f,g⟩² - ⟨f,f⟩⟨g,g⟩ ≤ 0
  2. |f(x)|² = ⟨k(·,x),f⟩² ≤ k(x,x)⟨f,f⟩, lo que implica que si ⟨f,f⟩ = 0 entonces f es idénticamente cero
Definición. Para un (compacto) 𝒳 ⊆ ℝᵈ, y un espacio de Hilbert ℋ de funciones f: 𝒳 → ℝ, decimos que ℋ es un Espacio de Hilbert con Núcleo Reproducente si ∃k: 𝒳 → ℝ, tal que:
  1. k tiene la propiedad reproductora: f(x) = ⟨f(·), k(·,x)⟩
  2. k genera ℋ = span{k(·,x): x ∈ 𝒳}

Teorema de Mercer

Teorema 1.1 (Mercer). Suponga que k es un núcleo continuo semidefinido positivo en un conjunto compacto 𝒳, y el operador integral Tₖ: L₂(𝒳) → L₂(𝒳) definido por
(Tₖf)(·) = ∫𝒳 k(·,x)f(x)dx
es semidefinido positivo, es decir, ∀f ∈ L₂(𝒳),
∫𝒳 k(u,v)f(u)f(v)dudv ≥ 0
Entonces existe una base ortonormal {ψᵢ} de L₂(𝒳) que consiste en autofunciones de Tₖ tal que la secuencia correspondiente de autovalores {λᵢ} son no negativos. Las autofunciones correspondientes a autovalores no nulos son continuas en 𝒳 y k(u,v) tiene la representación
k(u,v) = ∑ᵢ₌₁ᵒᵒ λᵢ ψᵢ(u)ψᵢ(v)
donde la convergencia es absoluta y uniforme, es decir,
límₙ→∞ supᵤ,ᵥ |k(u,v) - ∑ᵢ₌₁ⁿ λᵢ ψᵢ(u)ψᵢ(v)| = 0
===== Página 4 =====

Para tomar una analogía en el caso finito, es decir, 𝒳 = {x₁,...,xₙ}. Sea Kᵢⱼ = k(xᵢ,xⱼ), y f: 𝒳 → ℝⁿ con fᵢ = f(xᵢ). Entonces,

Tₖf = ∑ᵢ₌₁ⁿ k(·,xᵢ)fᵢ
∀f, fᵀKf ≥ 0 ⇒ K ≽ 0 ⇒ K = ∑ₖ λₖ vₖ vₖᵀ

Por lo tanto,

k(xᵢ,xⱼ) = Kᵢⱼ = (VΛVᵀ)ᵢⱼ = ∑ₖ λₖ vₖᵢ vₖⱼ = ∑ₖ λₖ ψₖ(xᵢ)ψₖ(xⱼ) ⇒ ψₖ(xᵢ) = (vₖ)ᵢ

Resumimos varias condiciones equivalentes sobre k simétrico continuo definido en 𝒳 compacto:

  1. Toda matriz de Gram es semidefinida positiva
  2. Tₖ es semidefinido positivo
  3. k puede expresarse como k(u,v) = ∑ᵢ λᵢ ψᵢ(u)ψᵢ(v)
  4. k es el núcleo reproductor de un RKHS de funciones en 𝒳
